Interface for request handlers. Request handlers are functors which receive a Context and a callback as parameters.

The request handler may then transform the content of a BuildingResponse object based on the contents of an HttpRequest, both provided by the Context.

Request handlers can be asynchronous. When they are, don't forget to keep a [std::shared_ptr] around. You can get a handle on one of those by calling shared_from_this on the Context.

For more informations, check out the Request Pipeline Manual.
