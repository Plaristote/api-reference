The first argument of the default constructor may either be a pointer to a EVP_MD structure, which contains the implementation of a cipher. Examples be md5, sha, ripemd, etc.

You may also pass a string as parameter to MessageDigest's constructor. In which case, it will try to find a cipher using EVP_get_digestbyname.
